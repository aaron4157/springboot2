# springboot2

 JRE-17 執行環境的 java web 專案，引入 MyBatis 框架介接資料庫

## 相容性
- JDK 版本：17.0.3
- Spring boot 版本: 3.2.0
- Spring security 版本: 6.2.0
- 模板引擎: Thymeleaf 3.0.15
- MyBatis 版本: 3.2.14
- thymeleaf-layout-dialect: 2.4.1 (nz.net.ultraq.thymeleaf)
